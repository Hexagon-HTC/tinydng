# Simple DNG viewer example with bullet3's OpenGLWindow + ImGui.

## Requirements

* premake5
* OpenGL 2.x

## Build on Linux/MacOSX

    $ premake5 gmake
    $ make

## Build on Windows

T.B.W.

## Usage

    $ ./bin/native/Release/view /path/to/file.dng
